# Circular Area

## Quick start guide:
- Enable one of the three mapping types and apply the setting


## Explanation of the mappings:
**(u,v)** are circular coordinates in the domain {(u,v) | u² + v² ≤ 1}

**(x,y)** are square coordinates in the range [-1,1] x [-1,1]
